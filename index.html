QR Vidéos YouTube

Site Amélie live

Un studio QR simple pour les vidéos YouTube.

Collez une URL YouTube, ajoutez un nom si besoin, et le studio génère un QR code unique avec aperçu immédiat, téléchargements individuels en PNG et SVG, plus un export groupé.

Stockage

Conservé localement dans ce navigateur pour rester compatible avec un projet GitHub + Vercel.

Export

Le ZIP génère un dossier qrcodes/ avec les fichiers et un index.json.

Ce que permet l’outil

  • Validation des URLs YouTube
  • Prévisualisation instantanée
  • Téléchargement unitaire PNG / SVG
  • Historique des QR générés
  • Export ZIP complet
